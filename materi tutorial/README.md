# materi-tutorial

* Analisis Data (dalam bahasa R)
* MATLAB (dalam bahasa Octave)
* Manajemen Institusi Astronomi
* Matematika Numerik (dalam bahasa Octave)
* Simulasi dan Komputasi Matematika (dalam bahasa Octave dan Python)
